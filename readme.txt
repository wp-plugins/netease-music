网易云音乐歌单

== Description ==

网易云音乐歌单，可以同步你在网易云音乐的歌单并播放。


音乐资源来自网易云音乐music.163.com

== Installation ==

1. 上传 `neatease-music`目录 到 `/wp-content/plugins/` 目录
2. 在后台插件菜单激活该插件
3. 后台填写你的网易云音乐用户ID
4. 直接使用短代码[nm]网易云音乐用户ID[/nm]，或者在需要的地方使用插入netease_music();
